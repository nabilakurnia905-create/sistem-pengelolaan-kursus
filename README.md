# Sistem Pengelolaan Kursus Berbasis Web

Aplikasi pengelolaan kursus berbasis web menggunakan PHP Native dan MySQL.

## Fitur
- Login pengguna
- Role Staff
  - Pendaftaran
  - Melihat detail data pendaftar
- Role Kepala Kursus
  - Kelola data siswa
  - Kelola data kursus
  - Kelola data pengajar
- CRUD (Tambah, Edit, Hapus Data)

## Teknologi
- PHP Native
- MySQL
- HTML, CSS, Bootstrap
- Laragon

## Database
File database tersedia dalam bentuk file `.sql`.

## Cara Menjalankan
1. Simpan project di folder `www` (Laragon)
2. Import database melalui phpMyAdmin
3. Jalankan melalui browser

## Catatan
Project dijalankan pada server lokal (localhost).
